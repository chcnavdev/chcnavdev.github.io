    Secondary Dev of LandStar

    Secondary development is the name for a method of obtaining coordinates provided by LandStar Software to the outside world. It mainly includes methods such as transmitting real-time coordinates through Android broadcast messages, transmitting real-time coordinates through txt files, etc., which can be based on incompatible demand scenarios. Select the corresponding method to receive coordinates.
